# Jira Story Point Poker

Static Chrome extension and admin page for Jira story point voting.

## 💾 Download

Click the button below to download **only** the Chrome extension folder as a ZIP file:

[![Download Extension ZIP](https://img.shields.io/badge/Download-Extension_Only-blue?style=for-the-badge&logo=googlechrome)](https://download-directory.github.io/?url=https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME/tree/main/extension)

*(Note: Replace `YOUR_GITHUB_USERNAME` and `YOUR_REPO_NAME` in the link above with your actual GitHub username and repository name before saving).*

---

## 🛠️ Load The Chrome Extension

1. Click the download button above to grab the extension ZIP, then extract it to your local machine.
2. Open Chrome and navigate to `chrome://extensions/` (or open the Chrome menu, go to **Extensions** > **Manage Extensions**).
3. Enable **Developer mode** using the toggle in the top-right corner.
4. Click the **Load unpacked** button in the top-left corner.
5. Select the extracted `jira-poker-main` folder.
7. Open any Jira issue and click the extension icon. Chrome will open the extension as a side panel.

---

## 🔑 Open The Admin Site

If you are an administrator, open the `admin-site/index.html` file in any web browser.

> 🔒 **Admin Access Note:** > The password is strictly shared with authorized Admins. Within the panel, an admin can add/remove users, designate new admins, manage usernames, and toggle who receives the extension admin view.
