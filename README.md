# Jira Story Point Poker

Static Chrome extension and admin page for Jira story point voting.

## 💾 Download

Click the button below to download the latest version of the extension and admin site as a ZIP file:

[![Download ZIP](https://img.shields.io/badge/Download-Repository_ZIP-blue?style=for-the-badge&logo=github)](../../archive/refs/heads/main.zip)


---

## 🛠️ Load The Chrome Extension

1. Download and extract the repository ZIP file to your local machine.
2. Open Chrome and navigate to `chrome://extensions/` (or open the Chrome menu, go to **Extensions** > **Manage Extensions**).
3. Enable **Developer mode** using the toggle in the top-right corner.
4. Click the **Load unpacked** button in the top-left corner.
5. Select the `jira-poker-main` folder from the extracted repository files.
6. Open any Jira issue and click the extension icon. Chrome will open the extension as a side panel.

---

## 🔑 Open The Admin Site

1. Navigate into the repository folder on your machine.
2. Open the `admin-site/index.html` file in any web browser.

> 🔒 **Admin Access Note:** > The password is strictly shared with authorized Admins. Within the panel, an admin can add/remove users, designate new admins, manage usernames, and toggle who receives the extension admin view.
