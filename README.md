# Jira Story Point Poker

Static Chrome extension and admin page for Jira story point voting.

## Load The Chrome Extension

1. Open `chrome://extensions` or can move to manage extensions.
2. Enable Developer mode.
3. Click "Load unpacked".
4. Select [jira-poker](/download/extension). ( open your folder location where extension is stored )
5. Open a Jira issue and click the extension icon. Chrome opens the extension as a side panel.

## Open The Admin Site

Open [admin-site/index.html](/Users/yshukla/Documents/task-pointer/admin-site/index.html) in a browser. Password is only shared with the Admin who can add or remove users accordingly , admin can also setup different admins as well , then add usernames and toggle who receives the extension admin view.
